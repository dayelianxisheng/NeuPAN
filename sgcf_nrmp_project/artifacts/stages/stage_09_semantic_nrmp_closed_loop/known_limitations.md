# Known Limitations

- R1 detects only explicit RGB availability, age, projection validity, and UNKNOWN failures. Silent calibration drift remains undetectable.
- The first evaluation harness uses procedural oracle semantics; it is not a real image network.
- The blocked run exposed empty-scene geometry rejection and CPU latency regressions that need lifecycle profiling.
- The emergency-stop fixture begins in collision and therefore cannot coexist with a literal aggregate zero-collision assertion without separate metric semantics.
- The named scenario harness is preliminary; compound scenes and semantic-infeasible status handling require completion after the blocker is resolved.
